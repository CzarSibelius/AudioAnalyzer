# AudioAnalyzer.Visualizers

Visualization implementations. TextLayersVisualizer is the only `IVisualizer`. All visual content is implemented as `ITextLayerRenderer` layers (Marquee, BeatCircles, Oscilloscope, GeissBackground, VuMeter, LlamaStyle, etc.).

**Contents**: `IVisualizer` implementation (TextLayersVisualizer), `ITextLayerRenderer` layers (Marquee, ScrollingColors, BeatCircles, Oscilloscope, GeissBackground, UnknownPleasures, VuMeter, LlamaStyle, etc.), visualizer settings types

**Dependencies**: Application, Domain
