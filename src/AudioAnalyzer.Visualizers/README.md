# AudioAnalyzer.Visualizers

Visualization implementations. Contains standalone `IVisualizer` modes (Spectrum, Geiss, Oscilloscope, etc.) and `ITextLayerRenderer` layers for the Layered text mode.

**Contents**: `IVisualizer` implementations (SpectrumBars, VuMeter, WinampBars, Oscilloscope, Geiss, UnknownPleasures, TextLayersVisualizer), `ITextLayerRenderer` layers (Marquee, ScrollingColors, BeatCircles, etc.), visualizer settings types

**Dependencies**: Application, Domain
