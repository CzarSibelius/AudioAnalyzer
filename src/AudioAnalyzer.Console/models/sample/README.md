# Sample Wavefront OBJ models

These `.obj` files are copied next to the executable on build. Use them with the **AsciiModel** text layer:

1. Open preset settings (**S**), select a layer, set type to **AsciiModel**.
2. Set **Model folder** to the full path of this directory, or a path relative to how you launch the app (e.g. `models\sample` when running from the build output folder).

Press **I** to cycle between `cube.obj` and `tetrahedron.obj` (alphabetical order).
